# draggavue

Lightweight, type-safe, and fully accessible drag & drop primitives for Vue.js.

> [!NOTE]
> Work in progress. API documentation will land here as features are merged.

## Goals

- **Tiny & fast** — zero dependencies, pointer-event driven, `transform`-based rendering
- **Fully type safe** — strict TypeScript, generics all the way down
- **Accessible by default** — keyboard dragging, screen reader announcements, WAI-ARIA
- **Headless & customizable** — composables first, unstyled components on top
- **Simple to adopt** — install, import, drag
